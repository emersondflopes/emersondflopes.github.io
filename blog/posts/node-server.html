How to Create a Simple NodeJS Web App

21 April, 2018

To start the project open your terminal and run npm init -y, this will start a project for us.

We're going to use Express for this project npm install express this will install express, you can see you on the package.jsonfile that the first command created, on the dependences.

Now we need to create a new JavaScript file on your terminal run touch app.js it will create a new this file, then open it on your text editor.

On this file we will put the following code:

In the first line const express = require('express'); we are importing express from the node_modules folder. then on the second we are involking the express function as app

On line 4 we are setting up our routes, this is the url you visit on the web. In example, on google.com/ this is the root route, this will be the same for our app, we are setting the root route by putting '/'. We're doing a GET request, when you visit the root route the code on the next line will be executed.

Still on line 4 we're passing an arrow function with req which stands for request and res which stands for response. We're making use of the response argument, and we're sending 'hello world'.

On line 8 we're telling node which port we want our app to run. On every computer there are thousands of ports you can accept and send requests, in our app we're telling node to run on port 3000. It will run on our local server: 127.0.0.1

So if now we just need to run our app. In the terminal run node app.js, the code on line 9 will be executed.

Now you just need to make a GET request on your localhost. We can do that in the browser, go and in the adress bar type 127.0.0.1:3000, you will get the message 'hello world'.

There we go, we have our very simple express app!
